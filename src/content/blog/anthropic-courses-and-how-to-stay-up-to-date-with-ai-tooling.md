**TL;DR**

These are essential even if you use OpenAI or another LLM provider instead of Claude — they're general foundations for working with any generative AI, not Claude-specific:

- [AI Fluency: Framework & Foundations](https://anthropic.skilljar.com/ai-fluency-framework-foundations) — a framework for how and when to work with AI
- [AI Capabilities and Limitations](https://anthropic.skilljar.com/ai-capabilities-and-limitations) — the right mindset for what AI can and can't do

If you're more technical and want to build: [Building with the Claude API](https://anthropic.skilljar.com/claude-with-the-anthropic-api).

Below — my take on the other courses. Pick what's relevant to you, skip the rest.

---

1. [Claude 101](https://anthropic.skilljar.com/claude-101) — short, good starting point if you're new to Claude products. ([certificate](https://verify.skilljar.com/c/wk6ncq2yx743))
2. [Claude Code 101](https://anthropic.skilljar.com/claude-code-101) — short, but too surface-level to use Claude Code effectively. ([certificate](https://verify.skilljar.com/c/rkir46bmc4ys))
3. [Introduction to Claude Cowork](https://anthropic.skilljar.com/introduction-to-claude-cowork) — good, quick intro to Cowork. Recommended for everyone. ([certificate](https://verify.skilljar.com/c/at3gdkdragov))
4. [Claude Code in Action](https://anthropic.skilljar.com/claude-code-in-action) — pretty outdated, avoid it. ([certificate](https://verify.skilljar.com/c/wutikz4rpwuv))
5. [AI Fluency: Framework & Foundations](https://anthropic.skilljar.com/ai-fluency-framework-foundations) — great general course, a framework for how and when to work with AI. Recommended for everyone. ([certificate](https://verify.skilljar.com/c/ne6d3hq87tfq))
6. [Building with the Claude API](https://anthropic.skilljar.com/claude-with-the-anthropic-api) — detailed course covering MCP, agents and workflows, basic RAG, evals, and prompt engineering. Recommended for any builder or engineer. ([certificate](https://verify.skilljar.com/c/rpctxjntke5k))
7. [Introduction to Model Context Protocol](https://anthropic.skilljar.com/introduction-to-model-context-protocol) — covered inside Building with the Claude API. Skip if you took that one. ([certificate](https://verify.skilljar.com/c/abjmd2r66g84))
8. [AI Fluency for educators](https://anthropic.skilljar.com/ai-fluency-for-educators) — watched because I work in EdTech. Practical application of AI Fluency: Framework & Foundations for course and lesson design. ([certificate](https://verify.skilljar.com/c/fnqpxnt7zfov))
9. [AI Fluency for students](https://anthropic.skilljar.com/ai-fluency-for-students) — AI Fluency: Framework & Foundations in practice for learners. Core idea: use AI as a learning partner, not a substitute for actually learning. ([certificate](https://verify.skilljar.com/c/iz9umac2wsbr))
10. [Model Context Protocol: Advanced Topics](https://anthropic.skilljar.com/model-context-protocol-advanced-topics) — detailed, focuses on the HTTP approach for MCP. Worth a look if you want to build your own MCP server. ([certificate](https://verify.skilljar.com/c/utx2xneum75k))
11. [Claude with Amazon Bedrock](https://anthropic.skilljar.com/claude-in-amazon-bedrock) — same content as Building with the Claude API but for Bedrock. Pick one based on your platform. ([certificate](https://verify.skilljar.com/c/9o6njq32v6yo))
12. [Claude with Google Cloud's Vertex AI](https://anthropic.skilljar.com/claude-with-google-vertex) — same content as Building with the Claude API but for Vertex AI. Pick one based on your platform. ([certificate](https://verify.skilljar.com/c/o2zwsexnxbuf))
13. [Teaching AI Fluency](https://anthropic.skilljar.com/teaching-ai-fluency) — good reference if you need to teach others about AI fluency. ([certificate](https://verify.skilljar.com/c/m5g72j8ucu2c))
14. [AI Fluency for nonprofits](https://anthropic.skilljar.com/ai-fluency-for-nonprofits) — 4D framework applied in practice. Useful well beyond the nonprofit context. ([certificate](https://verify.skilljar.com/c/qsytw8kh9vpc))
15. [Introduction to agent skills](https://anthropic.skilljar.com/introduction-to-agent-skills) — short, good intro to skills. Key points: skills priority, and the idea that skills should ideally be invoked automatically — Claude uses semantic matching on the skill's description to decide when to trigger it. Note: subagents only use skills that are explicitly listed for them. ([certificate](https://verify.skilljar.com/c/tpt8bbqmucqj))
16. [Introduction to subagents](https://anthropic.skilljar.com/introduction-to-subagents) — short, good intro to subagents. Covers when to use them and when not to. ([certificate](https://verify.skilljar.com/c/i9k7edgewqyz))
17. [AI Capabilities and Limitations](https://anthropic.skilljar.com/ai-capabilities-and-limitations) — must-have. Sets the right mindset for what to keep in mind when working with AI. ([certificate](https://verify.skilljar.com/c/s5bv9ez7z7zd))

**Not covered in the courses but worth learning:**

- [Set up a useful status line](https://docs.claude.com/en/docs/claude-code/statusline) — customize the Claude Code status bar to show context (model, cost, branch, etc.)
- [Claude Code auto mode](https://claude.com/blog/auto-mode) — skip permission prompts safely; a classifier reviews each action before it runs

**Important:** these courses are just a baseline — Claude and other AI tools evolve every few weeks. To stay current, I'd use Claude Cowork (or other tool) to pull and digest content from [X.com](https://x.com/). For whatever reason, X is where Anthropic and OpenAI engineers post their thoughts and practices, and where companies drop their release notes.
